Frågor och svar om webbutveckling

Vad är ett element, attribut och värde?

Ett element är en del av HTML-dokumentet som definierar och strukturerar innehållet på webbsidan. Ett attribut ger extra information om ett HTML-element och värden är det som tilldelas attributen.

Vad är HTTP och vad innebär ett kommunikationsprotokoll?

HTTP står för HyperText Transfer Protocol och är ett protokoll som används för att överföra data över webben mellan webbservrar och webbläsare. Ett kommunikationsprotokoll är ett regelverk som styr kommunikationen mellan datorer.

Vem är W3C och vad är deras huvudsakliga uppgift?

W3C står för World Wide Web Consortium och är en organisation som arbetar för att utveckla standarder och riktlinjer för webben för att främja dess långsiktiga tillväxt.

Vad står WWW för och vad är detta?

WWW står för World Wide Web och är en global samling av resurser och dokument som är tillgängliga över Internet.

En typisk internetadress kan se ut på följande vis http://www.facebook.com/home.php

  • Vad kallas denna internetadress på fackspråk (en förkortning)?
  • URL
  • Vad betyder denna förkortning?
  • Uniform Resource Locator
  • Vad kallas a http://, b www.facebook.com och c /home.php?
  • a) Protokoll (HyperText Transfer Protocol)
  • b) Domännamn
  • c) Sökväg

Vad använder vi kommunikationsprotokollet FTP till?

FTP står för File Transfer Protocol och används för att överföra filer över Internet.

Vilken modell bygger HTTP på?

HTTP bygger på klient-server-modellen.

Vad står HTML för och vilket typ av språk är detta?

HTML står för HyperText Markup Language och är ett märkspråk för att strukturera innehåll på webbsidor.

Varför använder vi oss av indentering när vi skriver HTML-kod?

Indentering används för att göra koden lättläslig och för att organisera den hierarkiskt.

Blir det någon skillnad på utseendet av sidan när du indenterar din HTML-kod?

Nej, indentering påverkar inte utseendet på webbsidan för användaren, det påverkar endast läsbarheten för utvecklaren.

Varför lär vi oss HTML-kod från grunden?

Att lära sig HTML-kod från grunden gör att vi förstår hur webbsidor fungerar och kan skapa mer anpassade och flexibla lösningar.

Vilka två delar består ett HTML-dokument av?

Ett HTML-dokument består av <head>-delen och <body>-delen. I <head>-delen finns metadata och länkar till externa resurser medan <body>-delen innehåller själva innehållet som visas på sidan.

Varför har vi HTML-kommentarer i vår kod?

HTML-kommentarer används för att lägga till anteckningar eller förklaringar i koden som inte syns för användaren men som kan vara användbara för utvecklare när de arbetar med koden.
